สำหรับผู้เริ่มต้น Python Number

การทำงานกับ number (ตัวเลข) ใน Python เป็นสิ่งพื้นฐานที่ผู้เริ่มต้นควรเข้าใจ เนื่องจากตัวเลขมีบทบาทสำคัญในการคำนวณ การจัดการข้อมูล และการดำเนินการต่างๆ ในโปรแกรม Python

ประเภทของข้อมูลตัวเลขใน Python มีด้วยกัน 3 ประเภทดังนี้:

ตัวอย่างการใช้งาน int

ประเภทข้อมูล Integer (int) ในภาษา Python คือข้อมูลตัวเลขที่เป็นจำนวนเต็ม (positive, negative, or zero) โดยไม่มีจุดทศนิยม

ตัวอย่างการใช้งาน float

float หมายถึงตัวเลขทศนิยม ซึ่งสามารถใช้ในกระบวนการคำนวณต่างๆ ที่ต้องการค่าทศนิยม เช่น การหารที่ให้ค่าทศนิยม การใช้ค่าในฟังก์ชันทางคณิตศาสตร์ หรือการจัดการข้อมูลที่มีทศนิยม

ตัวอย่างการใช้งาน complex

complex คือ ประเภทข้อมูลที่ใช้เก็บจำนวนเชิงซ้อน (complex numbers) ซึ่งประกอบด้วย ส่วนจริง (real part) และ ส่วนจินตภาพ (imaginary part) โดยจำนวนเชิงซ้อนมักใช้ในคณิตศาสตร์, ฟิสิกส์, และวิศวกรรม
